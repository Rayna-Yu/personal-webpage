  • Nim Sum Player

    • Nim Sum Player is an interactive computer program that plays the classic mathematical strategy game Nim. The program leverages the binary properties of the XOR operation to calculate the Nim-sum of all heaps at each turn. By analyzing the bitwise representation of heap sizes, it determines an optimal move that transforms the current game state into a "zero Nim-sum" position— a provably safe state that guarantees eventual victory under perfect play. In essence, the program encodes the winning strategy discovered by C. Bouton in 1901.

  • Smart Image Shrinker

    • Smart Image Shrinker is a smarter way of shrinking an image. Instead of cropping and losing important information, or squishing which distorts the image, this program will calculate the "seam" with the least energy (least significant) seam in an image and deletes it. This program uses dynamic programing to optimize the run time in calculating the energy of each seam in the image.
